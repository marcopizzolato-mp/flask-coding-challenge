Raster Upload

This an exploratory tool, that allows the user to inspect a remotely sensed (raster) image with few simple clicks.

The user can upload a raster image of choice, with the tool returning both the raster metadata information,
(such as coordinate system, bounding box, statistics, etc.) and a raster thumbnail image.

Instructions:

1. Select the raster in .tif format using the Choose File button

2. Upload the raster with the button Submit

3. The tool will redirect automatically to a new page File Info

4. Two new pages Metadata and Thumbnail will be also made available to the user

Note:

In this current version (v1.0) the tool can only analyse single band rasters in .tif format.
