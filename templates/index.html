Raster Upload

This tool is designed to allow a simple interaction between the user and a remotely sensed image.

The tool provides some information on the uploaded file as well as a thumbnail of the asset, and relevant metadata
such as coordinate system, bounding box, statistics, etc.

Please select the Raster using the Choose File button, and upload it with the button Submit, once you do so
you will be redirected to a new page.

Note: for the scope of this exercise we are looking only at single band Rasters in .tif format.
